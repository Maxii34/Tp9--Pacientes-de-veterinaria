# 🐾 Administrador de Citas - Veterinaria

Aplicación para gestionar citas de mascotas. Permite agregar, ver y eliminar citas. Los datos se guardan en `localStorage`.

---

## 🛠️ Tecnologías utilizadas

- React
- React Hook Form
- React Bootstrap
- Bootstrap Icons
- SweetAlert2
